## Run Locally

Clone the project

```bash
  git clone https://github.com/Desp0o/lineDev-ToDo-App.git
```

Go to the project directory

```bash
  cd lineDev-ToDo-App
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
